# React Email Template

This project shows an example of how you can create a template for your emails with pure react and send them with SMTP.js.

The technologies used in this article are:

- [Next.js](https://nextjs.org/)
- [Smtp.js](https://smtpjs.com/)
- [Chakra UI](https://chakra-ui.com/)
- [React Hook Form](https://react-hook-form.com/)

[You can learn how to create this project in my blog](https://blog.imjulian.com/create-email-templates-with-react)
